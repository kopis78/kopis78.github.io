启用thinkspace主题的代码块行号

1.首先在_config.yml中关闭rouge渲染

_config.yml修改 
highlighter: none

2.然后禁用_layouts/default.html中的layout: compress，这个似乎存在bug

_layouts/default.html 
 ---
<!-- layout: compress -->
published: true
---

3.去prismjs官网选择主题、渲染语言和插件下载js和css，可以放到assets文件夹目录下，并在_include/head.html里添加引用

<script src="/assets/js/prism.js"></script>
<link rel="stylesheet" href="/assets/css/prism.css">

行号是依赖于line-numbers类，所以需要在_include/footer.html添加js来依次给代码块添加该类，js来自参考链接

<script>
var pres = document.getElementsByTagName("pre");
for(var i = 0; i < pres.length; i++){
    var pre = pres[i];
    if(pre.childNodes[0].nodeName == "CODE"){
        pre.setAttribute("class","line-numbers");
    }
}
</script>

4.如果此时无法显示行号，需要修改assets/css/prism.css，来增加行号的样式

assets/css/prism.css 
pre[class*=language-].line-numbers.line-numbers {
  padding-left: 3.8em !important; /* 恢复左侧留白 */
}

pre[class*=language-].line-numbers.line-numbers code {
  padding-left: 0 !important; /* 移除代码区域的留白 */
}

pre[class*=language-].line-numbers.line-numbers .line-numbers-rows {
  left: -3.8em !important; /* 行号列向左偏移到留白区域 */
}

大功告成！
